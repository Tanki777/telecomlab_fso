All CAD designs that were printed and used for this project are in this folder.
.step files are not editable and can be uploaded on a 3D printing software if some copies need to be made.
.f3d files are editable, if some changes to the original CAD drawings need to be done.
All pieces need M2.5 screws and nuts.

Brief explanation of the files in this folder:
- distance_10_cm is the cylinder that is fixed in front of the transmitter to prevent approaching an eye to the laser. It must be printed twice (the drawing
  is of half the cylinder).
- receiver_lens_15mm is the receiver support for the Thorlabs mountings. It consists of two pieces, if the lens is needed, it must be placed between the two
  parts (the 'flat' side towards the piece that is fixed to PCB, the convex side toward the piece that is placed on top).
- transmitter_lens_15mm follows the same principle of the receiver_support_15mm, for this project we did not use the lens on the transmitter side, but it could
  be useful for future experiments.
- receiver_support_txclose and transmitter_support_rxclose are flat components that can be fixed to the PCBs together and to each other if the laser OPV310
  (divergence 24°) is used, to keep the laser diode and the photodiode almost attached to each other.
